Projects

Pokemon BMI Calculator

I created a body mass index (BMI) calculator for Pokemon characters using a Pokemon API. The application was made using Python. Users can select from either Pikachu, Squirtle or Charizard and the program will tell the user the character's BMI and whether they are underweight, overweight or healthy.

Sweet Shop Management Database

I created a database for a Sweet Shop containing three tables: sweets, orders and couriers. The database allows the shop keeper to manage the stock of sweet products, add orders and look up courier details for each order. I created this database using MySQL.

Football League

I created a football league where users can view players and find out news or information on the league and players. I created this using Python and MySQL.
